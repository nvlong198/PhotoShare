# PhotoShare

Mạng xã hội chia sẻ ảnh

## Công nghệ sử dụng

- Front-end: ReactJS + Redux + Bootstrap
- Backend: Express.js
- Search engine: Elasticsearch
- Microsoft Cognitive Face API Service
- MySQL, Firebase

### Cài đặt
Sửa cấu hình kết nối DB tại ./config/config.json
```bash
npm install
npm start
```
